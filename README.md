# 🌿 康养老人体征管理平台

一个基于 Vue3 + Tailwind CSS 的养老院老人体征状态综合管理平台，采用温柔疗愈的康养风格设计。

## ✨ 项目特点

### 🎨 设计理念：康养风格
- **温柔疗愈配色**：草木青 #80C4A8、生命绿 #6EC28C、舒缓蓝 #7AB6E9
- **圆角卡片布局**：柔和阴影、大量留白、低对比度
- **情感化表达**：使用 Emoji 头像、心情状态、暖色告警
- **非工业化设计**：弱化科技蓝/黑色、减少锐角和金属线框

### 📊 核心功能

#### 1. 实时体征监控
- ❤️ 心率监测
- 🫁 血氧饱和度
- 🌡️ 体温监测
- 🩸 血压监测
- 😴 睡眠质量分析
- 🚶 活动量统计

#### 2. AI 健康分析
- 🤖 AI 实时分析老人健康状态
- 📈 7日/30日健康趋势预测
- 🔍 异常模式识别
- 💡 智能健康建议
- 📊 可信度评分显示

#### 3. 智能告警系统
- 🚨 危险告警（红色、脉冲动画）
- ⚠️ 预警提示（橙色、柔和闪烁）
- ℹ️ 一般提醒（蓝色）
- 📍 定位到具体房间
- 🔔 实时浮层通知

#### 4. 楼层地图可视化
- 🏢 5层楼房间分布
- 🎨 房间状态色彩编码
- 👤 每个房间的老人信息
- 📊 健康评分可视化
- 🔍 点击查看详情

#### 5. 个人健康档案
- 📋 完整的个人信息
- 💓 实时体征数据
- 📈 历史趋势图表
- 😴 睡眠质量详情
- 📱 设备状态监控

#### 6. 设备管理
- 📱 智能手表
- 📡 红外监测
- 💓 心电监护
- 🔋 电量监控
- 📶 在线状态

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:5173

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 🎨 设计系统

### 配色方案
| 名称 | 颜色值 | 用途 |
|------|--------|------|
| 康养主色 | #80C4A8 | 主色调、按钮、重点数据 |
| 生命绿 | #6EC28C | 正常体征、健康指数 |
| 舒缓蓝 | #7AB6E9 | 数据详情、分区背景 |
| 暖橙 | #FFA45B | 轻度告警 |
| 健康红 | #FF6B6B | 紧急告警 |
| 象牙白 | #FBFBF5 | 背景 |
| 柔灰 | #404040 | 文本 |

### UI组件特点
- 圆角卡片：1-2rem 圆角
- 柔和阴影：0 2px 12px rgba(0,0,0,0.06)
- 温柔过渡：0.3s cubic-bezier(0.4, 0, 0.2, 1)
- 呼吸动画：3s pulse-slow
- 悬浮效果：轻微上浮 + 阴影增强

## 🔧 技术栈

- **框架**：Vue 3.4 (Composition API)

- **CSS 框架**：Tailwind CSS 3.4
- **图表库**：ECharts 5.5
- **路由**：Vue Router 4.3
- **日期处理**：Day.js 1.11
